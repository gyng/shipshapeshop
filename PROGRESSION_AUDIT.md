# Progression audit — Shape Gacha vs. the best idle games

Benchmarks: Cookie Clicker, AdVenture Capitalist, Antimatter Dimensions, Realm Grinder, NGU Idle, Melvor Idle.

## What Shape Gacha already does well
- **Two-currency loop** — Flux (idle production) → pulls; Shards (duplicates) → Forge + Relic summon. Good conversion loop.
- **Closed-form offline catch-up** (capped, generous) — a genre staple, done right.
- **Prestige / New Game+** — `recrystallize` ascends a dimension, ×1.6 global, carries the collection, adds an NG+ content cohort.
- **Layered multipliers** — prestige × Platonic-set (+15%) × bonds × kin-synergy (+8%/pair). Multiplicative stacking is the right shape.
- **Scarcity that isn't a paywall** — the Euler-budget loadout is a genuinely novel "what do I deploy" decision.
- **Finite + completable by design** (~1–2 days, NG+ extends) — a deliberate, healthy divergence from infinite treadmills.

## Gaps vs. genre hallmarks (highest-impact first)

1. **No upgrade shop.** The #1 missing hallmark. Every great idle has a long list of *one-time permanent purchases* that structure the mid-game and give Flux a deep, satisfying sink beyond gacha. Right now Flux only sinks into pulls + cosmetics.
   → **Add an Upgrades tab**: Expand Floor (Euler cap +2 ×N), Raise Production Cap, Faster Bonds (affinity rate), Bigger Offline Cap (12h→24h→48h), Cheaper Pulls / kinder pity. Curated + finite (it's not an infinite tree), each with a clear cost curve.

2. **Prestige is a flat ×1.6, not a choice.** AdCap angels / Realm Grinder factions / NGU give a *spendable meta-currency + perk tree* on reset.
   → **Recrystallize grants "Facets"** spent on permanent cross-run perks (start with N Flux, +X% global, a free first deploy, keep one Shop scene, +1 starting Euler). Turns NG+ into build variety, not just a bigger number.

3. **No milestones/achievements → permanent bonuses.** Antimatter Dimensions/Melvor lean on these for both dopamine and power.
   → **Milestones**: own 10/25/41 shapes, forge 5 recipes, Bond-5 any shape, deploy 3 kin pairs, summon all Relics — each a small permanent global bonus + a visible checklist (Zeigarnik). Surface in the Ledger.

4. **Production hard-caps at 900/hr** → the late run plateaus (no "number go up" near completion).
   → **Scale the cap with prestige** (e.g., cap × prestige_mult, or a flat per-NG bump) so each ascension visibly escalates while staying bounded/finite.

5. **Thin automation.** Auto-arrange exists (manual); idling is passive.
   → **Auto-pull toggle** (spend Flux when full) and **auto-forge** (discovered recipes) — quality-of-life for long idles, unlocked as upgrades/milestones.

6. **No explicit objective tracker.** The Nudge shows one hint; players lack a "what's my next goal" board.
   → A small **Objectives panel** (or expand the Ledger) listing the next 2–3 concrete goals with progress bars.

## Recommended order
**Upgrade shop** (1) → **Milestones** (3) → **prestige Facets** (2) → **cap scaling** (4) → **automation** (5) → **objectives UI** (6).
All keep the cozy, finite, no-dark-patterns spirit: bounded trees, transparent costs, no FOMO/timers.
